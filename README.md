# CraftingManipulator

API used to modify crafting results in crafting table


For build.gradle (1.21):
```groovy
  repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    modImplementation 'com.github.Xires87:CraftingManipulator:5.2.0'
}
```
